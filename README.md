# SwiftTradeUI

SwiftTradeUI is a modern React-based Trade Finance application providing secure, role-based access to Letters of Credit, Bank Guarantees, Collections, Trade Loans, workflows, approvals, dashboards, and transaction monitoring.

## Tech Stack

- React
- TypeScript
- Vite
- Redux Toolkit
- Material UI
- React Router
- Axios

## Features

- Authentication & Authorization
- Role-Based Access Control
- Trade Finance Dashboard
- Transaction Monitoring
- Workflow Approvals
- Responsive UI
